Tictactoe is a simple and very weak tic tac toe engine coded in C#. 

How to use:
Enter your number(0-8, upper left to bottom-right)
AI will reply.
Play till someone wins.
Test branch is at time of writing (3.7.2026) very much not working yet, but the basics are there. If you want a strong and usable engine, please don't use this one. This is only a preparation for me to write a chess engine, so this engine here will be very inefficient.
